# React + TypeScript + Vite + GraphQL + Express

## To run the frontend

1. cd into the frontend folder
2. run `npm install`
3. run `npm run dev`
4. open [http://localhost:5173/](http://localhost:5173/) in your browser

## To run the backend

1. cd into the backend folder
2. run `node server.ts`

You can also test the backend by opening [http://localhost:4000/graphql](http://localhost:4000/graphql) in your browser
